# Myers-Briggs Classification
Machine Learning Classification for Myers-Briggs personality test.

Project for ML exam.
## How to run
conda:
```bash
conda env create -f ML_requirements.yml
```
```bash
conda activate ML2
```

```sh
python3 main.py
```
